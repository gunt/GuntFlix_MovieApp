GuntFlix Movie API - Documentation

REST API for “GuntFlix MovieApp” that interacts with a database that stores data about different movies. Will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.
Business Logic URL HTTP METHOD Request body data format Response body data format
Get a list of ALL movies to the user /movies GET None JSON object holding data about all movies.
Get data about a single movie by title to the user /movies/:Title GET None JSON object holding data about the description, genre, director, image URL of a movie.
Get data data about a movie by genre (description) /movies/genres/:Title GET None JSON object holding data about a movie genre, containing name and description.
Get data about a director by name /movies/directors/:Name GET None JSON object holding data about a director by name.
Create a new User /users POST A JSON object holding data about the user to add - structured: { Username: paulinaGomez Password: paulina007 Email: paulinagom_007@gmail.com Birthdate: 05/11/83 } A JSON object holding data about the user created - structured: { Username: paulinaGomez Password: paulina007 Email: paulinagom_007@gmail.com Birthdate: 05/11/83 }
Update the user info (Username, Password, Email, Birthdate) /users/:Username PUT None A text message showing the user's information was updated.
Add a movie to the User's favorites list /users/:Username/Movies/:MovieID POST None A text message indicating the movie title was added to the user's favorites list.
Remove a movie from user's favorites list /users/:Username/Movies/:MovieID DELETE None A text message indicating the movie title was deleted from the user's favorites list.
Allow existing users to delete his User Profile /users/:Username DELETE None A text message was showing that user was deregistered successfully.
