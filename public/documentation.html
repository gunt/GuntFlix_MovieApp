GuntFlix Movie API - Documentation

REST API for “GuntFlix MovieApp” that interacts with a database that stores data about different movies. Will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.
Business Logic URL HTTP METHOD Request body data format Response body data format
Get a list of ALL movies to the user /movies GET None JSON object holding data about all movies.
Get data about a single movie by title to the user /movies/[title] Get None JSON object holding data about the description, genre, director, image URL of a movie.
Get data data about a movie by genre (description) /movies/genres/[genre] GET None JSON object holding data about a movie genre, containing name and description.
Get data about a director by name /movies/directors/[name] GET None JSON object holding data about a director by name.
Create a new User /users POST A JSON object holding data about the user to add - structured: { username: paulinaGomez password: paulina007 email: paulinagom_007@gmail.com date_of_birth: 11.11.1983 } A JSON object holding data about the user created - structured: { username: paulinaGomez password: paulina007 email: paulinagom_007@gmail.com date_of_birth: 11.11.1983 }
Update their user info (username, password, email, date of birth) /users/[username]/[password]/[email]/[date_of_birth] PUT None A text message showing the user's information was updated.
Add a movie to the User's favorites list /favorites/[username]/[title] POST None A text message indicating the movie title was added to the user's favorites list.
Remove a movie from user's favorites list. /favorites/[username]/[title] DELETE None A text message indicating the movie title was deleted from the user's favorites list.
Allow existing users to deregister/th> /users/[username] DELETE None A text message was showing that user was deregistered successfully.
